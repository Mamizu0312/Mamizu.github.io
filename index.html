Minecraft Plugin

DropProbability

使用言語: Java ,対応バージョン: 忘れた
ブロック破壊時のアイテムドロップ率が調節できるプラグインです。
大学受験が終わったあとのリハビリで作ったものですが、ちゃんと動いたプラグインはこれが初なんじゃないだろうかと思います。
ただ更新はしてないのでやっぱり産業廃棄物
github: DropProbability

SimpleSpawn

使用言語: Java, 対応バージョン: 忘れた(超古い)
/spawnコマンドを実装するやつです。configファイルで座標を設定できます。
例によって2018年に作ったブツなのでもちろん最新のバージョンでは動かないです。
github: SimpleSpawn
